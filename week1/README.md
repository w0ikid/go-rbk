# Text Processor

CLI-инструмент на Go для обработки текстовых файлов по заданным правилам.

## Запуск

```bash
go run ./cmd input.txt output.txt
```

## Правила обработки

**Команды** применяются к предыдущим словам:

| Команда | Действие |
|---|---|
| `(hex)` | Конвертирует предыдущее слово из hex в десятичное |
| `(bin)` | Конвертирует предыдущее слово из binary в десятичное |
| `(up)` | ВЕРХНИЙ РЕГИСТР |
| `(low)` | нижний регистр |
| `(cap)` | Первая Буква Заглавная |
| `(up, N)` / `(low, N)` / `(cap, N)` | Применяет преобразование к N предыдущим словам |

**Пунктуация** — знаки прилипают к предыдущему слову:
```
hello , world  →  hello, world
wait !? ok     →  wait!? ok
```

**Артикли** — `a` заменяется на `an` перед гласными и `h`:
```
a apple  →  an apple
a hotel  →  an hotel
```

## Структура проекта

```
├── cmd/main.go          — точка входа
├── internal/
│   ├── processor.go     — основная логика
│   ├── transforms.go    — up / low / cap / hex / bin
│   ├── punctuation.go   — правила пунктуации
│   └── articles.go      — замена a → an
└── go.mod
```
